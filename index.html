IKEA 3D Model Downloader

Download 3D models from IKEA for Sweet Home 3D and other software

ℹ️ How to Use

  1. Paste an IKEA product URL below (e.g., from ikea.com/de/de or ikea.com/no/no)
  2. Click "Process" to start the extraction
  3. If the 3D model doesn't load automatically, you'll see a message asking you to click "View in 3D" on the IKEA page
  4. Once the GLB file is found, it will be downloaded
  5. Download the model as a ZIP file with OBJ format and textures

⚠️ Note: This browser version has limitations. For best results, use the command line tool: npm run test:integration

Example: https://www.ikea.com/de/de/p/poaeng-schaukelstuhl...
Supported: 🇩🇪 Germany, 🇳🇴 Norway, 🇸🇪 Sweden, 🇬🇧 UK, 🇺🇸 USA, 🇫🇷 France, and more

💡 Troubleshooting Tips:

  • Make sure you're using a valid IKEA product URL
  • Try using a product from Germany (ikea.com/de/de) - they work most reliably
  • For best results, use the command line: npm run test:integration
  • Check the browser console (F12) for detailed error messages

✅ Privacy: No data is sent to external servers. All processing happens in your browser.
